\section{Component Extraction}

% ===================== 7 CIRCLES =====================
\subsection{7 Circles}

\subsubsection{Technique Used and Justification}

The image contains circular components. The following pipeline was used:

\begin{itemize}
    \item Gaussian Blur: reduce noise
    \item Otsu Thresholding: separate objects from background
    \item Morphological Operations: clean segmentation
    \item Contour Analysis: detect circular shapes using circularity
\end{itemize}

This method was chosen because circularity-based detection is robust for simple geometric shapes.

\subsubsection{Code}

\begin{lstlisting}[language=Python]
gray = to_gray(img)
blur = gaussian_blur(gray, 5)
thresh = otsu_threshold(blur)
opened = morph_open(thresh, 3, 1)
closed = morph_close(opened, 5, 1)
\end{lstlisting}

\subsubsection{Results}

\begin{figure}[H]
\centering
\includegraphics[width=0.8\linewidth]{../data/processed/componentExtraction/7circles/comparison.png}
\caption{7 Circles Processing}
\end{figure}

\subsubsection{Discussion}

\textbf{Pros:}
\begin{itemize}
    \item Accurate circle detection
    \item Robust to small noise
\end{itemize}

\textbf{Cons:}
\begin{itemize}
    \item Sensitive to thresholding
\end{itemize}

\textbf{Future Work:}
Use Hough Circle Transform for better robustness.

    \item Unsharp Masking
